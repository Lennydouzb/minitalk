# Description

Minitalk aim at creating a communication between a client and a server using UNIX signals

# Instructions

To compile the project :

```make
make
```
To use the program :

Launch the server first
```bash
./server
```

Use the printed pid in the cient

```bash
./client <pid> <string>
```
